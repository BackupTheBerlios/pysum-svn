PYSUM(1)
=========
Daniel Fuentes B. <dbfuentes@gmail.com>

NAME
----
pysum - A simple pygtk application for create md5 and other checksum

SYNOPSIS
--------
*pysum*

DESCRIPCION
-----------
pysum is a simple pygtk application for create and verify md5,
sha-1, sha-256 and other checksum. Written in python using PyGTK
and it's licensed under the GPL.
